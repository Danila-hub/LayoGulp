Need to add in (F1 => Open Settings [JSON]):


"path-autocomplete.pathMappings": {
        "@img": "${folder}/src/img",
        "@scss": "${folder}/src/scss",
        "@js": "${folder}/src/js",
    }

And download an extension:
Path Autocomplete
